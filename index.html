Never get lost anymore!

Introducing Inertial Navigation System. A navigation system that doesn't need any external stimulus.

Easy to use

This system can be used anywhere, everywhere, as long as you're still on Earth.

Low Battery

This system only use some sensors that need little internet and use background process with little memory

No Internet

Inertial Navigation System doesn't need any external stimulus to work. You just need to record the starting location and there it is!

Pros and Cons using this system

ADVANTAGES
  1. Easy to use
  2. Doesn't any connection to internet
  3. Low battery use
DRAWBACKS

Low accuracy
This navigation system still struggle with precise mathematical calculation because this needs to calculate double numeric integration which will deviate in a long run.
